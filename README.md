# CTA Crime Analysis 2022-2025

**Live dashboard:**
The interactive Streamlit dashboard: [here](https://cpqbkxxgjuxwabwamt2kjq.streamlit.app)

## Project Overview
This project analyzes how crime exposure near Chicago CTA L stations relates to ridership patterns over time 2022-2025. Using station-level ridership data, geocoded crime incidents, and spatial joins, we produce static visualizations to inform how the City of Chicago and CTA might better allocate transit security resources to improve transit utilization.

## Data Sources

| Dataset | Description | Source |
|---|---|---|
| `CTA_Ridership_L_Station_Entries_Daily_Totals_2022-2026.csv` | Daily ridership entries per station | Chicago Data Portal |

## Setup

**1. Clone the repo and set up the conda environment:**

```bash
conda env create -f environment.yml
conda activate ctacrime
```

**2. Download the crime data** from the [project Dropbox](https://www.dropbox.com/scl/fo/sq360chzws1wlj3o2qyop/AATdS7xIi-0s0bwWrSTLXgI?rlkey=xgfa2m0xq61bxsp7ogfxhpiu0&st=jy69syqm&dl=0) and place the CSV in `data/raw-data/`.

**3. Run preprocessing** (one time вЂ” takes ~1вЂ“2 minutes):

```bash
python streamlit-app/preprocessing.py
```

This performs a spatial join: for each station, it counts crime incidents within a **450m radius** per month and merges that with monthly ridership totals. Output files are saved to `data/derived-data/`.

**4. Launch the dashboard:**

```bash
streamlit run streamlit-app/app.py
```
